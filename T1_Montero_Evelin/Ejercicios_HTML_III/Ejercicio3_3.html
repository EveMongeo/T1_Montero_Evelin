Ejercicio 3.3

Complete las siguientes oraciones:

  1. El atributo autofocus (que se utiliza en un solo elemento input en un formulario) resalta de manera automática el elemento input y, si se usa de manera apropiada, coloca el cursor en el campo de texto después de que el navegardor carga y despliega la página.
  2. Los nuevos tipos de HTML5 tienen validación automática del lado del cliente.
  3. El tipo input number permite al usuario introducir un valor numérico.
  4. El tipo input range se valida automáticamente de manera intrínseca cuando lo despliega el navegador como un control deslizante, ya que el usuario no puede mover el control fuera de lso límites de valor mínimo o máximo.
  5. El atributo autocomplete puede usarse en los tipos input para completar de manera automática la información del usuario con base en datos de entrada anteriores.
  6. El elemento figcaption proporciona una leyenda para la imagen en el elemento figure.
  7. El elemento summary muestra una flecha que apunta a la derecha enseguida de un resumen o leyenda cuando el documento se despliega en un navegador. Al hacer clic en la flecha, ésta apunta hacia abajo y revela el contenido del elemento details.
  8. El elemento mark nos permite resaltar texto.

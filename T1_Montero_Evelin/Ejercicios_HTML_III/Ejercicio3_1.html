Cuestionario 3.1

Complete las siguientes oraciones:

  1. El tipo input color permite al usuario introducir un color. Al momento de escribir este libro, la mayoría de los navegadores despliegan el tipo input color como un campo de texto en donce el usuario puede introducir un valor hexadecimal.
  2. El atributo placholder nos permite colocar texto temporal en un campo de texto.
  3. Si desea anular la validación, puede agregar el atributo formnovalidate al tipo input submit.
  4. El atributo required obliga al usuario a introducirun valor antes de enviar el formulario.
  5. Por lo general se muestra el control spinner para el tipo input number e incluye sólo números válidos.
  6. El tipo input time permite al usuario introducir una hora, minuto, segundo y fracción de segundo.
  7. El elemento datalist proporciona opciones de entrada para un elemento input text.
  8. El elemento header describe el contenido que está separado del contenido principal de la página y que posiblemente podría utilizarse o distribuirse en otra parte, como un artículo de noticias, mensaje de foro o entrada de blog.
  9. El elemento footer describe el texto que por lo general aparece en la parte inferior del contenido o de un elemento section.
  10. El elemento wbrindica el lugar apropiado para dividir una palabra cuando el texto se ajusta en varias líneas.
